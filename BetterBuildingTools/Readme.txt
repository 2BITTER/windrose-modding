BetterBuildingTools (BBT)
========================

Precision building tools for Windrose by 2BIT.

Install:
  1. Install UE4SS v3.x (https://github.com/UE4SS-RE/RE-UE4SS)
  2. Copy BetterBuildingTools folder to: Windrose\R5\Binaries\Win64\ue4ss\Mods\
  3. Make sure enabled.txt contains "1"
  4. Launch the game

Folder Structure:
  -Dev-/      Working source code (latest, may be unstable)
  Released/   Last published source (matches Nexus/CurseForge release)

Source files:
  src/        C++ source (8 files) — builds to DLL via CMake + UE4SS SDK
  Scripts/    Lua scripts (4 files) — UI, settings, HUD

Links:

  CurseForge: https://www.curseforge.com/windrose/mods/betterbuildingtools
